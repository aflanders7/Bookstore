# Bookstore Database Management App
Author: Audrey Flanders

[Link to Demo](https://www.youtube.com/watch?v=OGN5wG_WY0o)

# Built With
* MySQL
* Node.JS
* Express
* Handlebars
* JavaScript

# Getting Started
**1. Clone the repo**

    git clone https://github.com/aflanders7/Bookstore.git
  
**3. Install packages**

    npm install
   
**5. Run the program**

    node app.js
